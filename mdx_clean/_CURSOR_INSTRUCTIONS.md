# Writing Section — MDX Posts for gileslamb.com
## What's in this folder

29 MDX posts converted from Substack, April–December 2025.
Ready to drop into the Next.js project as-is.

---

## Step 1 — Copy files into project

Place all `.mdx` files (not this instructions file) into:
```
/content/writing/
```
Create the folder if it doesn't exist.

---

## Step 2 — Install MDX dependencies

```bash
npm install gray-matter next-mdx-remote
```

---

## Step 3 — Cursor prompt to build the section

> "Add a writing section called 'The Quiet Room' to the site using MDX files in `/content/writing/`.
>
> Use `gray-matter` to parse frontmatter and `next-mdx-remote` to render content.
>
> Build two things:
>
> **1. `/writing` index page**
> — List all articles in reverse chronological order (newest first)
> — Show: title, subtitle, formatted date
> — Section heading: 'The Quiet Room'
> — Same design system as rest of site: dark background (#080808), Cormorant Garamond titles, Karla body, cream text (#ede5d8), accent gold (#c9a96e) for dates/labels
> — Cards separated by a thin line, not boxed
> — Add 'Writing' link to main navigation
>
> **2. `/writing/[slug]` individual article pages**
> — Clean long-form reading layout
> — Title: Cormorant Garamond, ~3rem, weight 300
> — Subtitle: Karla italic, muted cream, below title
> — Date: small, gold (#c9a96e), below subtitle
> — Body: Karla, 1.05rem, line-height 1.9, max-width 660px, centred
> — Images: full width within content column, rounded corners
> — Back link: ← The Quiet Room (top left, small)
> — Same dark background throughout"
